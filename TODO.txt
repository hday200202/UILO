UILO TODO & Roadmap
===================

## High-Priority Widgets

- [ ] Calendar / Date Picker
      - Range selection mode
      - Month/year navigation
      - Keyboard shortcuts

- [ ] File Browser
      - Floating window mode (for dialogs)
      - Docked mode (embedded in panels)
      - File filtering by extension
      - Multi-select mode
      - Favorites / Recent files
      - Search/filter bar

- [ ] Color Picker
      - HSV/RGB selector
      - Gradient picker
      - Palette presets
      - Alpha channel control

- [ ] Tree View
      - Hierarchical display with expand/collapse
      - Lazy-load children on expand (virtualization for large trees)
      - Multi-level nesting support
      - Icons or custom content per node
      - Selection modes (single, multi, range)
      - Keyboard navigation (arrows, home/end, expand/collapse with left/right)
      - Optional drag-and-drop between nodes
      - Context menu integration
      - Scrollable when exceeds bounds

- [ ] Tabs / Tab Bar
      - Closeable tabs
      - Active tab highlighting
      - Overflow handling (scroll or menu)

## Features & Infrastructure

- [ ] Palette System (for application-defined theming)
      - Centralized color palette definitions
      - Palette switching at runtime
      - NO built-in themes; apps define their own palettes
      - Palette serialization/deserialization

- [ ] Style System
      - CSS-like selectors or style sheets
      - Reusable style objects
      - Cascading properties

- [ ] Animation System
      - Lerp between two or more Modifiers over time
      - Lerp between two or more Options over time
      - Easing functions (linear, ease-in, ease-out, etc.)
      - Duration and playback control
      - Design question: How does element know when to animate?
        * Manual trigger (.animate(startModifier, endModifier, duration))?
        * Callback-driven?
        * Event-based?
      - Callbacks on animation start/end

- [ ] Drag and Drop
      - Draggable elements
      - Drop targets
      - Visual feedback during drag
      - Mouse class with "bucket" for arbitrary payload data
        * Store any type of data during drag (void* or template-based)
        * Query payload at drop target
        * Enables flexible, type-safe data passing

- [ ] Keyboard Navigation
      - Tab order management
      - Focus indicators
      - Arrow key navigation in containers

- [ ] Virtual Scrolling
      - Lazy rendering for massive lists
      - Performance for 10k+ items

- [ ] Hot Reload / Live Edit
      - UI state preservation on reload
      - Fast iteration during development

- [ ] Inspector / Debug Tools
      - Visual element hierarchy
      - Property viewer
      - Performance profiler
      - Event logger

- [ ] Native File Dialog Abstraction
      - Unified interface for file open/save dialogs
      - Works across macOS, Linux, Windows
      - Path filtering and file type restrictions
      - Async or blocking modes
      - Returns selected path(s) and metadata

- [ ] Platform-Specific Features
      - OS-level drag & drop into UILO window
      - Cut/copy/paste integration with system clipboard
